## Скачать

Актуальные APK — в **[GitHub Releases](https://github.com/ONE1TOUCH-App/one1touch-android-app/releases/latest)**.

| Файл | Для кого |
|------|----------|
| `ONE1TOUCH-APP-1.0.6a-release-universal-arm.apk` | **Рекомендуется** (~121 MB) |
| `ONE1TOUCH-APP-1.0.6a-release-arm64-v8a.apk` | Только новые устройства (~76 MB) |

Перед установкой разрешите установку из выбранного источника (файловый менеджер, Telegram и т.п.).

Если у вас уже стоит более ранняя сборка 1.0.6a — удалите её перед установкой обновлённого APK из этого релиза.

## Быстрый старт

1. Скачайте **universal-arm** APK из Releases.
2. Установите и откройте **ONE1TOUCH**.
3. Войдите в аккаунт и следуйте подсказкам в приложении.
4. При первом подключении Android запросит разрешение — подтвердите.

На Xiaomi / ColorOS / HyperOS: **Настройки → Приложения → ONE1TOUCH** — включите автозапуск и снимите ограничения батареи.

## Что нового в 1.0.6a

- **Проксирование по приложениям** — в **Параметры** режим «Только выбранные».
- **Автопереключение подписок** — при сбое подключения переход к следующей подписке или ключу.
- **In-app обновления** — проверка новых версий в **Параметры**.
- Обновлённый личный кабинет (как в Windows 1.0.6).
- **Quick Settings** — плитка ONE1TOUCH в шторке, запуск последней конфигурации даже без открытого приложения.
- Мультиподписки, сохранение входа после перезапуска.

## Исправления (актуальная сборка релиза)

- Плитка в шторке корректно запускает подключение при «холодном» старте приложения.
- Стабильное восстановление авторизации после перезапуска.
- Отключение соединения с одного нажатия.
- Вкладка **«Ключи»** в «Подписки» переключается корректно.
- Полный список приложений в выборе для проксирования.
- Анимация **«ПОДКЛЮЧЕНИЕ...»** при нажатии кнопки подключения.
