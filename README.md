## Скачать

Актуальные APK — в **[GitHub Releases](https://github.com/ONE1TOUCH-App/one1touch-android-app/releases/latest)**.

| Файл | Для кого |
|------|----------|
| `ONE1TOUCH-APP-1.0.5a-release-universal-arm.apk` | **Рекомендуется** — все телефоны (Samsung, Xiaomi, Pixel, OPPO…) |

Обновление с **1.0.4**: установите новый APK поверх — вход в аккаунт и настройки сохранятся.

---

## Быстрый старт

1. Скачайте **universal-arm** APK из Releases.
2. Установите и откройте **ONE1TOUCH**.
3. Войдите в аккаунт и следуйте подсказкам в приложении.
4. При первом подключении Android запросит разрешение VPN — подтвердите.

На Xiaomi / ColorOS / HyperOS: **Настройки → Приложения → ONE1TOUCH** — включите автозапуск и снимите ограничения батареи.

## Что нового в 1.0.5

- **Мультиподписка** — несколько серверов в одной подписке и общая ссылка для импорта.
- Вкладки **«Подписки»** и **«Ключи»** в личном кабинете и при выборе подключения.
- **Сборщик подписки** — объединение ключей в одну подписку.
- Актуальный **статус серверов** и обновлённый личный кабинет (как в Windows 1.0.5).
- Плитка **ONE1TOUCH** в шторке быстрых настроек.
- Сохранение входа в аккаунт между запусками.
