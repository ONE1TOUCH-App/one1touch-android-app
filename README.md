## Скачать

Актуальные APK — в **[GitHub Releases](https://github.com/ONE1TOUCH-App/one1touch-android-app/releases/latest)**.

| Файл | Для кого |
|------|----------|
| `ONE1TOUCH-APP-1.0.6a-release-universal-arm.apk` | **Рекомендуется** — все телефоны (Samsung, Xiaomi, Pixel, OPPO…) |
| `ONE1TOUCH-APP-1.0.6a-release-arm64-v8a.apk` | Только новые устройства (меньше размер файла) |

Перед установкой разрешите установку из выбранного источника (файловый менеджер, Telegram и т.п.).

Обновление с **1.0.5a**: **Параметры** → «Проверить обновления» или установите новый APK поверх — вход и настройки сохранятся.

---

## Быстрый старт

1. Скачайте **universal-arm** APK из Releases.
2. Установите и откройте **ONE1TOUCH**.
3. Войдите в аккаунт и следуйте подсказкам в приложении.
4. При первом подключении Android запросит разрешение — подтвердите.

На Xiaomi / ColorOS / HyperOS: **Настройки → Приложения → ONE1TOUCH** — включите автозапуск и снимите ограничения батареи.

## Что нового в 1.0.6a

- **Проксирование по приложениям** — в **Параметры** режим «Только выбранные»: выбор из списка установленных приложений.
- **Автопереключение подписок** — при сбое подключения переход к следующей подписке или ключу.
- **Обновлённый кабинет** — паритет с desktop 1.0.6: модалки, продление bundle, инструкции.
- **Обновления в приложении** — **Параметры** → «Проверить обновления».
- **Стабильность** — вкладка «Ключи» в подписках, анимация при подключении, корректное отключение с одного нажатия.
